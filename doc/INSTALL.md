# silent-task-runner

A lightweight system tray application that silently schedules and executes commands at specified times.


## Install from PYPI

The homepage in pipy is https://pypi.org/project/silent_task_runner/

```bash
pip install --upgrade silent_task_runner
```

Using:

```bash
silent-task-runner
```

## Install from source
Installing `silent-task-runner` program

```bash
git clone https://github.com/trucomanx/SilentTaskRunner.git
cd SilentTaskRunner
pip install -r requirements.txt
cd src
python -m build
pip install dist/silent_task_runner-*.tar.gz
```
Using:

```bash
silent-task-runner
```

## Uninstall

```bash
pip uninstall silent_task_runner
```
